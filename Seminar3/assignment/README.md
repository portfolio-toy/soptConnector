### 2. Post 데이터 설계해보기

- 저희는 User, Profile Model을 가지고 있습니다.
- 마지막으로 게시판에 필요한 Post Model을 설계해봅시다.
- 화면이나 기능을 보고 어떤 데이터가 필요하겠다 생각하는 힘은 굉장히 중요하다고 생각합니다.
- 2차 세미나 프롤로그에서 봤던 게시판 뷰를 보면 게시글, 댓글, 좋아요 기능이 있는데 어떤 식으로 데이터를 설계해볼까 고민해보면 좋을 것 같습니다.
